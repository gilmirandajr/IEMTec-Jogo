# Guia do Usuário - Jogo de Inteligência Emocional

## Bem-vindo!

O **Jogo de Inteligência Emocional** é uma aplicação educacional interativa projetada para ajudar crianças e adolescentes (8-15 anos) a desenvolver suas habilidades socioemocionais através de atividades lúdicas e significativas.

## Powered by Manus

Este jogo foi desenvolvido com tecnologias modernas e de ponta:

- **Frontend:** React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui
- **Armazenamento:** localStorage para persistência de dados do usuário
- **Design:** Responsivo e acessível para todos os dispositivos (mobile, tablet, desktop)
- **Acessibilidade:** Suporte completo para navegação por teclado, leitores de tela e alto contraste
- **Deployment:** Aplicação web estática otimizada para distribuição rápida

## Usando Seu Jogo

O jogo oferece três módulos principais, cada um focado em diferentes aspectos da inteligência emocional:

### 1. Calendário Emocional

O **Calendário Emocional** permite que você registre e acompanhe suas emoções diárias. Ao acessar este módulo, você verá um calendário do mês atual com a seguinte funcionalidade:

- Clique em qualquer dia do calendário para registrar como você se sentiu
- Escolha entre seis emoções básicas: Feliz, Triste, Ansioso, Raiva, Calmo e Confuso
- Defina a intensidade da emoção em uma escala de 1 a 10
- Opcionalmente, descreva o motivo da emoção em um campo de texto
- Clique em "Registrar Emoção" para salvar seu registro
- Navegue entre meses usando os botões "←" e "→" para ver histórico
- Visualize seus registros anteriores clicando em "Ver Registro de Hoje"

Os dados são salvos automaticamente no seu navegador, permitindo que você acompanhe suas emoções ao longo do tempo.

### 2. Meditação Guiada

O módulo **Meditação Guiada** oferece técnicas de respiração e relaxamento para ajudar você a acalmar a mente e promover bem-estar. O jogo inclui quatro meditações diferentes:

- **Respiração 4-7-8:** Técnica de respiração que acalma o sistema nervoso (3 minutos)
- **Varredura do Corpo:** Relaxe cada parte do seu corpo conscientemente (5 minutos)
- **Meditação de Gratidão:** Cultive sentimentos positivos (4 minutos)
- **Respiração em Caixa:** Técnica para reduzir ansiedade (2 minutos)

Para usar:

- Selecione uma meditação clicando em "Começar Meditação"
- Siga as instruções passo a passo na tela
- Use os botões "Anterior" e "Próximo" para navegar entre os passos
- Clique em "Pausar" ou "Retomar" conforme necessário
- Quando terminar, clique em "Concluir Meditação"

### 3. Ouça os Sentimentos

O jogo **Ouça os Sentimentos** desenvolve sua empatia através de cenários interativos baseados em situações reais. Você enfrentará quatro cenários diferentes:

- **Seu Amigo Está Triste:** Aprenda a reconhecer quando alguém precisa de apoio
- **Colega Nervoso na Prova:** Pratique oferecer suporte emocional
- **Novo Aluno na Turma:** Desenvolva inclusão e acolhimento
- **Amigo Muito Bravo:** Aprenda a lidar com conflitos com empatia

Para cada cenário:

- Leia o contexto da situação
- Escolha uma das três opções de resposta
- Receba feedback imediato sobre sua escolha
- Veja sua pontuação de empatia (0-10)
- Avance para o próximo cenário

Ao final, você receberá uma pontuação geral de empatia que reflete como você respondeu aos cenários.

## Gerenciando Seu Jogo

### Dados Pessoais

Todos os seus registros (emoções, meditações, pontuações) são salvos localmente no seu navegador. Nenhum dado é enviado para servidores externos. Seus dados persistem enquanto você não limpar o histórico do navegador.

### Acessibilidade

O jogo foi desenvolvido seguindo os princípios do Design Universal para Aprendizagem (DUA):

- **Navegação por Teclado:** Use Tab para navegar entre elementos e Enter para ativar botões
- **Leitores de Tela:** Totalmente compatível com NVDA, JAWS e outros leitores de tela
- **Alto Contraste:** Cores cuidadosamente escolhidas para melhor legibilidade
- **Ajuste de Tamanho:** Use Ctrl + (Windows/Linux) ou Cmd + (Mac) para aumentar o tamanho da fonte

## Próximos Passos

Comece sua jornada de desenvolvimento emocional hoje mesmo! Dedique alguns minutos por dia para as atividades e observe como você melhora sua compreensão e gestão emocional.

**Dica importante:** Não há respostas certas ou erradas neste jogo. Cada sentimento é válido, e o objetivo é aprender a reconhecê-los e responder com empatia. Compartilhe suas experiências com amigos, família ou professores para aprofundar seu aprendizado!
